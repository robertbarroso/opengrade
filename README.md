# OpenGrade
An easy and simply open source solution for keeping track of student grades and assignments. Generate report cards to give to parents or guardians.

## Features
- Group students by category, subject, or age group.
- Have either weighted or unweighted homework, test, and quiz grades.
- User defined color-coded tags for assignment categories.
- Generate report cards for individual students.
- Analytic dashboard showing student performance by category, subject, or age group.
- All data is stored locally using SQLite.
